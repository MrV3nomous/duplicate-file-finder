# Elite Duplicate File Finder

A comprehensive CLI tool to find and manage duplicate files in any folder.

## Features

- Recursive scanning of folders and subfolders
- SHA256 hash-based duplicate detection for accuracy
- Interactive deletion keeping selected files
- Beautiful terminal table display with file sizes
- Progress bar for large directories
- Color-coded output for readability
- Efficient handling of large files
- Supports both small and large-scale directories
